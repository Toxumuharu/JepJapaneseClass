## Verbs for example
- たべる (to eat)
- のむ (to drink)
- あるく (to walk)
- およぐ (to swim)
- しはらう (to pay)
- はなす (to speak)
- よむ (to read)
- きく (to ask/listen/hear)
- すわる (to sit)
- にゅうりょくする (to type/input)
- べんきょう する (to study)
